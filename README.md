# Logo Creator

 ## Description
 This application allows users to create a logo using an SVG file.

 ## Installation
 - Node.js
 - Inquirer 8.2.4
 - Jest 24.9.0

 ## Usage
 As the user is prompted on the backend, answer the questions to choose your shape, text color, logo name, and logo color.
